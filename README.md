# Art Gallery E-Commerce Website

A responsive e-commerce platform for selling original artwork, featuring a complete shopping cart system and checkout functionality.

## Features

- **Shopping Cart**: Add, remove, and view items with real-time updates
- **Persistent Storage**: Cart data saved using localStorage, persists across sessions
- **Checkout System**: Complete shipping form with validation
- **Order Summary**: Real-time calculation of subtotal, shipping, and total
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage API
